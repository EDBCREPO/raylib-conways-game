# Conways Game Of Life
Conways Game Of Life using C++, Nodepp and Raylib

## Dependencies
```bash
📌Nodepp: https://github.com/NodeppOficial/nodepp
📌Raylib: https://www.raylib.com/
```

## Build & Use
```bash
time g++ -o main main.cpp -I ./include -L ./lib -lraylib -std=c++11 ; ./main
```
